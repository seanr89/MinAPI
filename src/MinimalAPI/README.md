# Minimal API
Basic API for testing minimal API and other settings etc...

In order to run:
```
dotnet run
```

navigate to swagger:
```
http://localhost:5008/swagger/index.html
```

## Structure
Basic API concepts to test out TodoItem storage, creation and reading
